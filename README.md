# Personal Portfolio Website

A modern 2026-style personal website with 3D animations, glassmorphism, and smooth scroll effects.

## Features

- **3D Background** - Floating geometric shapes and particle field (Three.js)
- **Glassmorphism UI** - Blurred glass-style cards
- **Responsive** - Works on mobile and desktop
- **Smooth Animations** - Scroll reveal and hover effects

## How to Run

1. Open `index.html` in a browser (double-click or use Live Server in VS Code)
2. Or run a local server: `npx serve .` from this folder

## Customize Your Content

Edit `index.html` to update:

| Section | What to Change |
|---------|----------------|
| Hero | Name, tagline, description, buttons |
| About | Bio text, highlights list |
| Skills | Skill categories and items |
| Projects | Project names, descriptions, links |
| Contact | Email, GitHub, LinkedIn URLs |

Your profile image is at `assets/profile.png` - replace it to update your photo.

## Tech Stack

- HTML5, CSS3, JavaScript
- Three.js (3D)
- Google Fonts (Outfit, Space Grotesk)
